## Instalación

1. Clonar este repositorio
   ```sh
   git clone https://github.com/Leoxl83/tiendaweb-langorta.git
   ```
2. Instalar NPM
   ```sh
   npm install
   ```
3. Iniciar la aplicación
   ```sh
   npm start
   ```
